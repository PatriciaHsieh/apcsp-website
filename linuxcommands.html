Linux Commands

Name of Command Description Example
uname Prints the system information uname -a = all information displayed
echo Echoes the provided string echo hello = displays hello
ls Lists information about the files ls = all files in the directory are displayed
date Displays the current time or set the system date date = Wed 1 May 19:03:44 UTC 2019
last Displays a list of all users logged in and out last = displays the number of times I logged in and when
clear Clears the screen clear = screen is cleared/becomes blank
who Print information about the users who are currently logged in who = sp2a
cal Displays a calendar and days until Easter cal 2002 = displays the calendar of 2002
mkdir Creates a directory in the destination provided mkdir test = creates a directory called test
sleep Pauses for the provided number of seconds sleep 5 = pauses for 5 seconds
